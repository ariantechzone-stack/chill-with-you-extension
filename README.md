# Chill With You ☕ Chrome Extension

Cozy focus room with **lofi music, rain sounds, and a Pomodoro timer**.

## Features
- Focus Mode: Play lofi music  
- Rain sounds: Play ambient rain  
- Adjustable volumes for both  
- Pomodoro timer (25 min focus) with bell alerts  

## How to Use
1. Download or clone this repository.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the `chill-with-you-extension` folder.
5. Click the extension icon → enjoy Focus Mode & Rain sounds → use the Pomodoro timer.

> ⚠️ Audio may require clicking once inside the popup to enable autoplay due to Chrome restrictions.

## Folder Structure
- `popup.html` → Popup UI  
- `popup.css` → Styling  
- `popup.js` → Logic (music, rain, Pomodoro)  
- `assets/audio/` → Audio files (lofi, rain, bells)  
- `icons/` → Extension icons  

## Notes
- No Chrome Web Store publishing required. Works locally on your browser.
